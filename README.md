# Rad51_RCNN
<h3>Nulcei, rad51 detector based on the mask-rcnn</h3> 

Input &nbsp;&nbsp; : <b> Stack of cell images </b><br>
Output : Detected & segmented nuclei and rad51s from cells along z-axis and its numbers in txt file.
<br>
<br>
